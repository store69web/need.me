## ⚡ Deployment Instructions

1.  **Initialize Interface:**
    Clone the repository and open `index.html`.
    ```bash
    git clone [https://github.com/diptarafd3t-cell/premium-bento.git](https://github.com/diptarafd3t-cell/premium-bento.git)
    ```

2.  **Configuration:**
    To update the payment numbers, modify the `copyNum` function parameters in the HTML body:
    ```html
    <div onclick="copyNum('YOUR_NUMBER_HERE', this)">
    ```

3.  **Responsive Testing:**
    The system includes breakpoints for **1050px** (Tablet) and **650px** (Mobile) to ensure the bento tiles stack correctly.
